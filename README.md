# Command Central — Deploy Guide

## Folder structure
```
command-central/
├── index.html       ← The app
├── css/style.css    ← Styles
├── js/app.js        ← Logic
└── data/sample.js  ← Sample data
```

## Deploy to Netlify (free, 2 minutes)
1. Go to https://netlify.com → sign up free
2. Click "Add new site" → "Deploy manually"
3. Drag the command-central folder onto the drop zone
4. Get your live URL instantly — bookmark it!

## Preview locally
Double-click index.html to open in browser.
